# Threadly — Social Media App

A clean, minimal social media platform built with **React** + **Bootstrap 5**.

## Features

- 🏠 **Home Feed** — Stories, compose box, tabbed For You / Following
- 🔍 **Explore** — Search posts with category filters
- 🔔 **Notifications** — Likes, follows, comments, reposts
- 🔖 **Bookmarks** — Save posts to read later
- 👤 **Profile** — Posts, replies, liked posts tabs
- ✍️ **Compose Modal** — Character counter, post creation
- 💬 **Inline Comments** — Expand comments per post
- ❤️ **Interactions** — Like, repost, bookmark, share
- 📱 **Responsive** — Mobile bottom nav, desktop sidebar

## Tech Stack

- React 18
- Bootstrap 5.3
- Bootstrap Icons
- DM Sans + DM Serif Display (Google Fonts)

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000)
